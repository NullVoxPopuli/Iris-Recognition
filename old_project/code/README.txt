This software was written by Justin Parks and Richard Solotke as part of a
term project for CSSE 490 - Image Recognition, Spring 2006.

Running our code is as easy as changing your working directory to the directory
in which this file resides and then running the script "main.m" by typing "main" at
the MATLAB command prompt.

Of course, you will need to place images in the same directory to make them available
to the script.  The images we used are available for download at:
http://phoenix.inf.upol.cz/iris/
